FAQ
Questions
answered
What tech is this built with? +
Pure HTML and CSS with a tiny sprinkle of vanilla JavaScript for interactions. No frameworks, no bundlers, no npm. Just open the file and it works.
Is it responsive on mobile? +
Yes — every section is tested across mobile (320px), tablet (768px), and wide desktop (1440px+). Columns collapse gracefully at each breakpoint.
Can I customise the colours? +
Absolutely. All colours are defined as CSS custom properties at the top of the file. Change 6 variables to completely retheme the entire page.
How do I host on GitHub Pages? +
Create a GitHub repo, push the HTML file, go to Settings → Pages, select your main branch. Your site is live in under 2 minutes.
Is there a free tier? +
Yes! The free tier includes 3 projects and access to all column templates. No credit card required.
